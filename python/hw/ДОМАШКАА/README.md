# Система управления студентами

Программа для управления данными о студентах и их средних оценках, реализующая различные типы методов Python.

## Описание

Программа демонстрирует использование:
- **Методов экземпляра** - для работы с конкретными объектами студентов
- **Методов класса** - для работы с коллекцией студентов
- **Статических методов** - для утилитарных функций

## Структура программы

### Класс `Student`
- `__init__(name)` - инициализация студента
- `add_grade(grade)` - добавление оценки (метод экземпляра)
- `get_average_grade()` - вычисление средней оценки (метод экземпляра)
- `display_info()` - отображение информации о студенте (метод экземпляра)
- `is_valid_grade(grade)` - проверка допустимости оценки (статический метод)
- `calculate_class_average(students)` - вычисление средней оценки всех студентов (метод класса)

### Класс `StudentManager`
- Управление списком студентов
- Поиск студентов по имени
- Добавление оценок студентам
- Отображение статистики

## Функциональность

1. **Добавление студентов** с именем и списком оценок
2. **Вычисление средней оценки** для каждого студента
3. **Вычисление средней оценки** всех студентов
4. **Проверка допустимости оценки** (от 1 до 5)
5. **Обработка ошибок** при вводе некорректных данных

## Запуск программы


### Демонстрационный режим

## Пример использования

```python
from student_manager import Student, StudentManager

# Создание менеджера
manager = StudentManager()

# Добавление студентов
manager.add_student("Иван Петров")
manager.add_student("Мария Сидорова")

# Добавление оценок
manager.add_grade_to_student("Иван Петров", 5)
manager.add_grade_to_student("Иван Петров", 4)

# Вычисление средней оценки
manager.display_student_average("Иван Петров")

# Средняя оценка всех студентов
manager.display_class_average()

# Проверка допустимости оценки
Student.is_valid_grade(5)  # True
Student.is_valid_grade(6)  # False
```

## Меню программы

1. Добавить нового студента
2. Добавить оценку студенту
3. Вывести среднюю оценку студента
4. Вывести среднюю оценку всех студентов
5. Проверить допустимость оценки
6. Показать всех студентов
7. Выход

## Обработка ошибок

- Проверка корректности оценок (1-5)
- Обработка некорректного ввода
- Проверка существования студентов
- Обработка пустых списков
